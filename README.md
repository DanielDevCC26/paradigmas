# 🐍 Exercícios de Python – Lógica de Programação

Este repositório contém uma coleção de exercícios desenvolvidos em Python com foco no aprendizado e prática de lógica de programação.

## 📚 Objetivo

O objetivo desta pasta é consolidar conhecimentos fundamentais da programação, por meio da resolução de exercícios práticos, abordando desde conceitos básicos até estruturas de controle.

## 🧠 Conteúdos abordados

- Operadores matemáticos
- Entrada e saída de dados (`input` e `print`)
- Tipos de dados
- Estruturas condicionais:
  - `if`
  - `elif`
  - `else`
- Estruturas de repetição:
  - `while`
- Lógica de programação
- Resolução de problemas

## 🛠️ Tecnologias utilizadas

- Python 3

## 📂 Organização

Os arquivos estão organizados de forma simples, com cada exercício representando um problema específico de lógica ou prática de sintaxe.

## 🚀 Aprendizados

Durante o desenvolvimento desses exercícios, foram trabalhadas habilidades como:

- Raciocínio lógico
- Interpretação de problemas
- Escrita de código limpo e organizado
- Prática constante com a linguagem Python
